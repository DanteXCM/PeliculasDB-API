# PeliculasDB-API

Este proyecto es una **API REST** diseñada para gestionar una base de datos de películas. Permite realizar operaciones de consulta y administración de contenido cinematográfico de manera eficiente.

## Descripción
PeliculasDB-API es un backend desarrollado para centralizar la información de películas, facilitando la integración con aplicaciones frontend o clientes de terceros. El proyecto se enfoca en la estructuración de datos y la entrega de respuestas en formato JSON.

## Funcionalidades
- **Gestión de Películas:** Consulta de títulos, géneros y detalles técnicos.
- **Arquitectura Escalable:** Estructura organizada para facilitar la expansión de rutas y modelos.
- **Formato Estándar:** Respuestas basadas en el estándar JSON para una comunicación fluida.

## Stack Tecnológico
* **Backend:** Python / Flask (o el lenguaje que estés utilizando en este parcial)
* **Base de Datos:** (Ej. MySQL / SQLite / JSON)
* **Herramientas:** Git, Visual Studio Code

## Estructura General
El repositorio sigue una estructura de carpetas estándar para aplicaciones de servidor:
- `/routes` o `App.py`: Definición de los endpoints de la API.
- `/models`: Lógica de la estructura de datos.
- `/data`: Archivos o scripts de base de datos.
